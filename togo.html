1.应用加固接口

请求URL: https://jaq.aliyuncs.com/

请求方式：GET

请求参数1：公共参数

名称 类型 是否必填 说明
Format String NO 返回值的类型，支持JSON与XML。默认为XML
Version String NO API版本号，为日期形式：YYYY-MM-DD，本版本对应为2016-04-12
AccessKeyId String NO 阿里云颁发给用户的访问服务所用的密钥ID
Signature String NO 签名结果串，关于签名的计算方法，请参见签名机制
SignatureMethod String NO 签名方式，目前支持HMAC-SHA1
Timestamp String NO 请求的时间戳。日期格式按照ISO8601标准表示，并需要使用UTC时间。格式为：YYYY-MM-DDThh:mm:ssZ。例如，2014-11-11T12:00:00Z（为北京时间2014年11月11日20点0分0秒）
SignatureVersion String NO 签名算法版本，目前版本是1.0
SignatureNonce String NO 唯一随机数，用于防止网络重放攻击。用户在不同请求间要使用不同的随机数值
RegionId String NO 仅支持 cn-hangzhou
Action String NO 调用接口的名称，如“Shield”

请求参数2： 接口参数

名称 类型 是否必填 说明
AppInfo JSON NO 待加固的应用信息,详细见下
Channel JSON NO 渠道列表，多渠道加固时填写
Enhance JSON NO 自定义加固信息，配置自定义加固时填写

AppInfo参数：

名称 类型 是否必填 说明
dataType JSON NO APP数据类型 1-App URL
data INT NO APP数据：dataType=1时填写 App包的下载地址
md5 string NO APP包的MD5值，dataType=1时必填，用于文件校验
size INT NO APP包大小（单位：字节），dataType=1时必填，用于文件校验
callbackUrl string NO 任务处理完成后的反向通知回调地址，dataType=1时必填，通知为GET请求，请求URL：callbackUrl+”?item_id=xxx&task_status=1”；item_id为应用加固接口返回的任务ID；task_status为任务状态：1-已完成,2-处理中,3-处理出错，4-处理超时；对于应用加固，接收到通知后如果 task_status 为1则可通过对应的查询接口查询加固结果
appOsType INT NO app 类型，1-apk

Channel参数：

名称 类型 是否必填 说明
metaName String NO AndroidManifest.xml 中用于表示渠道信息的meta-data标签的android:name
values List NO 渠道名称列表

Enhance参数：

名称 类型 是否必填 说明
enhanceType String NO 指定dex加固类型：0-不加固，1-轻量级加固，2-整体加固
javaAntiDex2Jar JSON NO 渠道名称列表
javaConstEncrypt JSON NO Java反dex2jar信息，配置java层常量加固时填写
soEnhance JSON NO so加固信息，配置自定义so加固时填写

javaAntiDex2Jar参数：

名称 类型 是否必填 说明
percent INT NO 指定反dex2jar的百分比配置

javaConstEncrypt参数：

名称 类型 是否必填 说明
percent INT NO 指定常量加固的百分比配置

soEnhance参数：

名称 类型 是否必填 说明
enhanceType String NO 指定so加固类型：0-不加固，1-轻量级加固，2-重量级加固
soFileList String NO 指定需要加固的so文件列表

请求实例：
https://jaq.aliyuncs.com/?Format=JSON
&Channel={"metaName":"channel","values":["91","360"]}
&SignatureMethod=HMAC-SHA1
&Signature=EYXamTa+RIafYaPoUdwHdwWpSvA=
&Timestamp=2016-06-05T06:41:30Z
&Enhance={'enhanceType': -1, 'javaAntiDex2jar': {'percent': 20},'javaConstEncrypt': {'percent': 20},'soEnhance': {'enhanceType': 1,'soFileList': ['lib/armeabi/libbitmaps.so','lib/armeabi/libgifimage.so']}}
&Action=DiyShield
&AccessKeyId=accessKeyId
&AppInfo={"appOsType":1,"callbackUrl":"http://aaa.com/callback","data":"http://g01.alibaba-inc.com/tfscom/LB1PaMeKXXXXXX8XFXXXXXXXXXX.tfsprivate1446115983140-375","dataType":1,"md5":"ce86f08da845d0af6d9df2a958de17b0","size":1713656}&**RegionId**=cn-hangzhou
&SignatureNonce=eac373bc-355f-4e9e-95fc-61cb986c8a80
&Version=2016-04-12
&SignatureVersion=1.0

返回参数：

名称 类型 是否必填 说明
ErrorCode INT YES 错误码
ErrorMsg string YES 错误说明
Data array YES 数据结果 数组

data数组：

名称 类型 是否必填 说明
itemId string YES 任务唯一标识,adef0394-3370-4e94-82c6-07af0d15a9cd，最少占用36字符
progress INT YES 任务处理进度：1-处理完成（可立即通过对应的查询接口查询处理结果），2-异步处理中（需等待 app_info.callback_url 接收到反向通知后再查询结果）；对于 app_info.data_type 为1目前都是异步处理，此字段返回2
